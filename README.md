# 💳 Program Card UI

A modern and interactive **Program Card** component built using HTML and CSS.  
This project focuses on clean UI design, smooth hover effects, and responsive layout.

## 🚀 Live Demo
👉 https://behrokh-dev.github.io/program-card/

## ✨ Features
- 🎨 Clean and modern UI design  
- ⚡ Smooth hover and transition effects  
- 📱 Fully responsive layout  
- 🧩 Reusable card component  
- 💡 Simple and lightweight (no frameworks)

## 🛠️ Built With
- HTML5  
- CSS3
